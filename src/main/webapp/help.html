DataCite Metadata Search Help

DataCite Metadata Search offers a simple search interface for metadata of datasets registered with MDS. Just type keywords in the search field and hit the enter key (or click on the Search button) to get a list of results.

This provides faceted search. It allows users to drill into search results by any value in designated fields. Each facet displayed also shows the number of hits within the search that match that category.

When you type multiple keywords the results will contain all metadata records where all keywords were present. If you wish to find the record which match any of the keywords you need to separate them with OR operator.

Also please note that the search is case insensitive.

Fields

Most of the fields in the DataCite Metadata Schema are searchable. Have a look at the Schema documentation for non-self-explanatory fields.

DataCite's mandatory fields

  • doi
  • creator
  • publisher
  • publicationYear
  • title

DataCite's optional fields

  • alternateIdentifier
  • contributor
  • date
  • description
  • format
  • language
  • relatedIdentifier
  • resourceType
  • resourceTypeGeneral
  • rights
  • size
  • subject
  • version

Other Fields

  • allocator: symbol of the DataCite member (e.g. BL = British Library)
  • datacentre: symbol of the datacentre, which uploaded the metadata
  • indexed: datestamp of indexing
  • prefix: DOI prefix
  • refQuality: reference quality flag
  • uploaded: datestamp of metadata uploading
